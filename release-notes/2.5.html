WindowSnap 2.5

  • Calendar events go to Outlook — "＋ Calendar Event" now creates the event directly in Microsoft Outlook when it's installed (falling back to the system Calendar otherwise), and asks for a title and duration first.
  • World clock in the menu bar — the menu now shows the current time in each city selected in the Convert tab's World Time grid, and your city choices survive relaunch.
  • Convert tab — World Time is now the default category, with Currency next in the list.
  • Settings backup — new Export/Import buttons in the Settings tab move every preference, shortcut, and saved layout to another Mac as a single JSON file.
  • Faster, sharper window switcher — thumbnails now load via ScreenCaptureKit, so the switcher opens instantly and live window images fill in as they arrive.
  • Under the hood: modernized macOS 14 APIs, a reorganized codebase (every file under 800 lines), a new unit-test suite, and CI on every change.
