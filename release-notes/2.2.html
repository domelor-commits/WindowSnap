WindowSnap 2.2

  • Redesigned the Layouts tab: each layout now has clear Restore and Overwrite buttons that work with a single click — no need to assign a keyboard shortcut first.
  • Overwriting a layout now asks for confirmation, so you can’t accidentally replace a saved arrangement.
  • Keyboard shortcuts moved into a tidy popover behind a ⌨ button, where you can set or clear the restore and overwrite hotkeys.
